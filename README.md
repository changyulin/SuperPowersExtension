# SuperPowersExtension

如果你习惯使用 SuperPowers 的 AI Coder，也可以直接用本仓库的 skills 来弥补 SuperPowers 在规则、经验沉淀和文档维护上的不足。
这个仓库用于补充 SuperPowers 中缺少的 skills，把可复用的工程经验整理成独立、可维护的能力模块。

## 当前 Skills

### `project-docs`
项目文档维护 skill，主要用于补充和更新 `README`、项目总览、架构说明、业务规则、编码规范、ADR、模块说明和 `how-to`。

### `experience-memory`
工程经验记忆 skill，用于检索、记录、晋升和清理可复用经验，支持项目内记忆和全局记忆两层沉淀，帮助在实现、排错、重构和补测试前先查经验再动手。
