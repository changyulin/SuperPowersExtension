# SuperPowersExtension

这个仓库用于补充和扩展 SuperPowers 风格的 skills，重点覆盖规则约束、工程经验沉淀、项目文档维护，以及代码审查与质量保障等场景。

这些 skills 可以配合现有工作流一起使用，也可以单独复用。

## 当前 Skills

### `project-docs`

项目文档维护 skill，主要用于补充和更新 `README`、项目总览、架构说明、业务规则、编码规范、ADR、模块说明和 `how-to`。

### `experience-memory`

工程经验记忆 skill，用于检索、记录、晋升和清理可复用工程经验，支持项目内记忆和全局记忆两层沉淀，帮助先查经验再动手。

### `code-review-quality`

代码审查与质量保障 skill，用于审查代码变更、PR、diff、重构和生成代码，重点关注正确性、安全性、可维护性、性能、测试缺口与回归风险。
